Vead

API tagastab vead OperationOutcome ressursina mille sees võib olla mitu issue elementi koos vigadega. Vead võivad olla kas ainult patsiendi registri omad või sisaldada ka peegeldatud vead teistest teenustest.

Näide OperationOutcome-ist, mis kuvab peegeldatud vead:

{
  "resourceType": "OperationOutcome",
  "issue": [
    {
      "code": "exception",
      "details": {
        "coding": [
          {
            "system": "https://mpi.tehik.ee",
            "code": "MPI-029"
          }
        ],
        "text": "Integratsiooniviga xroad-rr teenusega"
      }
    },
    {
      "code": "exception",
      "details": {
        "coding": [
          {
            "system": "https://xroad-rr.tehik.ee",
            "code": "XRR200"
          }
        ],
        "text": "X-road error 10001 from RR: Isikukoodi pikkus ei ole 11 sümbolit."
      }
    },
    {
      "code": "exception",
      "details": {
        "coding": [
          {
            "system": "https://rahvastikuregister.ee",
            "code": "10001"
          }
        ],
        "text": "Isikukoodi pikkus ei ole 11 sümbolit."
      }
    }
  ]
}
